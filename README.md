# OpenRC Podman Compose Service Scripts
OpenRC service scripts to run Podman Compose with a specified [Compose](https://compose-spec.io/) file. 

# Description
Use these OpenRC service scripts to manage execution of Podman Compose, given a `compose.yaml` file.
